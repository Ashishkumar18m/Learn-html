HTML List Tags: OL vs UL

<ol> - Ordered List (Numbered)

Creates a numbered list. Use when order matters.

Example:

  1. First step - Boil water
  2. Second step - Add pasta
  3. Third step - Cook for 10 minutes

<ul> - Unordered List (Bulleted)

Creates a bulleted list. Use when order doesn't matter.

Example:

  • Apples
  • Bananas
  • Oranges

Key Differences

  • OL displays numbers (1, 2, 3...)
  • UL displays bullets (• • •)
  • OL implies sequence/order matters
  • UL implies no specific order
